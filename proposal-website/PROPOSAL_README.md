2. **Open in browser**:
   Visit `http://localhost:3000`

3. **Share with your crush**:
   - You can deploy it online (see deployment options below)
   - Or show it to them in person on your device

## Customization Ideas

You can personalize the website by editing `app/page.tsx`:

- Change the main question text
- Add your crush's name
- Modify the colors in the gradient backgrounds
- Add more steps or messages
- Include inside jokes or special memories

## Deployment Options

### Vercel (Recommended - Free)
```bash
npm install -g vercel
vercel
```

### Netlify
```bash
npm run build
# Upload the 'out' folder to Netlify
```

### GitHub Pages
1. Push to GitHub
2. Enable GitHub Pages in repository settings
3. Deploy from the `gh-pages` branch
